## 🚀 How to Clone and Set Up This Project Locally

### 1️⃣ Prerequisites  
Before you begin, make sure you have:  
- ✅ **Git** installed ([Download Git](https://git-scm.com/downloads))  
- ✅ **Eclipse IDE** installed ([Download Eclipse](https://www.eclipse.org/downloads/))  
- ✅ **Access to the repository** (You must be added as a collaborator)  
- ✅ **GitHub Authentication**: If the repository is private and you have **2FA enabled**, you must use **SSH or a Personal Access Token (PAT)** instead of a password.

---

### 2️⃣ Clone the Repository  
#### 🔹 Option 1: Clone Using HTTPS (Recommended for Beginners)
1. Open a terminal (**Command Prompt, Git Bash, or PowerShell**).  
2. Run the following command:  
   ```bash
   git clone https://github.com/omarkhalod/Jackaroo.git
   ```

#### 3 If prompted for a password, use a Personal Access Token (PAT) instead of your GitHub password.
Generate a PAT here: https://github.com/settings/tokens

#### 4 After cloning, navigate to the project folder:
```bash
  cd Jackaroo
```
